# Code of Conduct

This project follows the Contributor Covenant 2.1 standard and adapts it for a repo where humans and AI agents both contribute.

## Our pledge

We pledge to make participation in this project respectful, harassment-free, and focused on the work. Contributors are expected to treat maintainers, users, reviewers, and other contributors with professionalism.

## Expected behavior

- Use welcoming and inclusive language.
- Respect different viewpoints and levels of context.
- Give and receive constructive technical feedback.
- Stay focused on what improves the project.
- Be clear when content was generated or substantially rewritten by an AI agent.

## Unacceptable behavior

- Harassment, insults, threats, or sustained disruption.
- Sexualized language or imagery.
- Public or private harassment.
- Publishing private information without explicit permission.
- Bad-faith use of AI-generated noise, spam, impersonation, or deceptive attribution.

## Enforcement

Report conduct concerns to `phsu@aotter.net`.

Maintainers may remove comments, close issues, block accounts, reject contributions, or take other action needed to protect the project. Enforcement decisions should be proportionate to impact, risk, and repeated behavior.

## Scope

This code applies in project spaces, including GitHub issues, PRs, discussions, security advisories, and any project-linked communication channels.
